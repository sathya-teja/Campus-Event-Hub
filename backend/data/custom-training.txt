## About CampusEventHub

CampusEventHub is an inter-college event management platform that connects students and colleges for various academic, cultural, technical, and sports events.

The platform supports:
- Event Discovery
- Online Event Registration
- Role-Based Access (Student / College Admin / Super Admin)
- Event Creation & Management (Admin Only)
- Participant Approval System
- Registration Tracking
- Dashboard Overview
- Multi-College Event Support

---

## User Types

1. Students
   - Create account, Login
   - Browse and register for events
   - Track registrations

2. College Admins
   - Login, Create and manage events
   - Approve or reject participants
   - View dashboard insights

3. Super Admin
   - Approve or reject college admin accounts
   - View all users and platform stats

4. Guests (not logged in)
   - Browse events
   - Must register/login to participate

## REGISTRATION GUIDANCE — ROLE AWARE

### If the user asks "How to register?" or "How do I register?":

- If they are ALREADY LOGGED IN (role is student, college_admin, or super_admin):
  → They are already registered. Do NOT show account creation steps.
  → Instead, tell them: "You're already registered and logged in! If you meant how to register for an event, here are the steps: ..."
  → Then show the event registration steps below.

- If they are a GUEST (not logged in):
  → Show the account creation steps.

### Event Registration Steps (for logged-in students):
1. Go to the Events section.
2. Browse available events.
3. Select the event you wish to join.
4. Review event details.
5. Click the Register button.
6. Confirm your registration.
7. Track your status in "My Registrations".

### Account Registration Steps (for guests only):
1. Go to the Register page.
2. Choose your role (Student or College Admin).
3. Enter your details (Name, Email, College, Phone).
4. Create a secure password.
5. Submit the form.
6. If Student — you can log in immediately.
7. If College Admin — your account needs Super Admin approval before you can log in.

## STUDENT GUIDANCE

### How to Browse Events:
1. Go to the Events section.
2. View available events.
3. Use filters (Category, Date) if needed.
4. Click an event to view details.

### How to Track Registrations:
1. Open your dashboard.
2. Navigate to "My Registrations".
3. View your registration status (Pending / Approved / Rejected).

## COLLEGE ADMIN GUIDANCE

### How to Create an Event:
1. Open the Admin Dashboard.
2. Click "Create Event".
3. Fill in: Event Name, Category, Description, Start Date, End Date, Location.
4. Upload an event image (optional).
5. Submit to publish the event.

### How to Manage Events:
1. Open the Admin Dashboard.
2. Go to "Event Management".
3. You will see all events you have created.
4. You can edit or delete your events from here.

---

## SUPER ADMIN GUIDANCE

### How to Approve a College Admin:
1. Open the Super Admin Dashboard.
2. Go to "Manage Admins".
3. View the list of pending college admins.
4. Click Approve or Reject for each request.

### How to View All Users:
1. Open the Super Admin Dashboard.
2. Go to "All Users".
3. Filter by role or search by name/email.
